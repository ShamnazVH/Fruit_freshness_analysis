# Fruit Category & Quality Classification (CNN)

## Project Overview
A deep learning project using CNNs to identify fruit types and assess their quality (fresh/rotten).

## How to Run
1. Install dependencies: `pip install -r requirements.txt`
2. Run the Flask app: `python app.py`

## Model Details
- **Architecture:** Convolutional Neural Network (CNN)
- **Framework:** TensorFlow/Keras
